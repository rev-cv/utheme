# Оформление блоков для Text Grid

Чтобы блоки в редакторе WordPress автоматически подхватывали стили из `text-grid.scss` и отображались как сетка карточек, следуйте инструкции ниже.

## Инструкция для редактора (Gutenberg)

1.  **Создайте контейнер (Блок "Группа")**
    *   Добавьте блок **Группа** (Group).
    *   В настройках блока справа, в разделе **Дополнительно** (Advanced) -> **CSS классы** (Additional CSS class(es)), добавьте класс: `text-grid`.

2.  **Добавьте карточки (Блоки "Список")**
    *   Внутрь созданной группы добавьте блок **Список** (List).
    *   Каждый отдельный блок списка будет одной карточкой.
    *   В настройках блока списка, в разделе **Дополнительно** -> **CSS классы**, добавьте класс: `text-grid__card`.

3.  **Наполните карточку**
    *   **Первый пункт списка**: Используйте для заголовка (выделите жирным **Bold**).
    *   **Второй пункт списка**: Используйте для описания.

## Пример HTML структуры

Если вы верстаете вручную или используете HTML-блок:

```html
<div class="wp-block-group text-grid">
    <!-- Внутренний контейнер (создается WordPress автоматически для групп) -->
    <div class="wp-block-group__inner-container">
        
        <!-- Карточка 1 -->
        <ul class="wp-block-list text-grid__card">
            <li><strong>Заголовок карточки</strong></li>
            <li>Текст описания карточки...</li>
        </ul>

        <!-- Карточка 2 -->
        <ul class="wp-block-list text-grid__card">
            <li><strong>Заголовок карточки 2</strong></li>
            <li>Текст описания карточки 2...</li>
        </ul>

    </div>
</div>
```
