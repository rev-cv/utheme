# Структура HTML для стилей cards.scss

Для того чтобы контент отображался в виде сетки карточек, соответствующих стилям в `cards.scss`, необходимо использовать следующую HTML-разметку.

## Пример HTML кода

```html
<div class="articles-container">
    <div class="articles-grid">

        <!-- Начало карточки -->
        <div class="card-page">
            <!-- Изображение (фон карточки) -->
            <img src="/path/to/image.jpg" alt="Описание изображения">

            <!-- Заголовок карточки -->
            <div class="card-page__title">
                Заголовок статьи или страницы
            </div>

            <!-- Ссылка (абсолютно позиционирована, покрывает всю карточку) -->
            <a href="/link-to-post" class="card-page__link"></a>
        </div>
        <!-- Конец карточки -->

    </div>
</div>
```

## Описание классов

*   **`.articles-container`** — Ограничивает ширину контента и добавляет внешние отступы.
*   **`.articles-grid`** — CSS Grid контейнер, который выстраивает карточки в сетку.
*   **`.card-page`** — Обертка отдельной карточки.
    *   **`img`** — Изображение записи. Стили растягивают его на всю высоту карточки (`object-fit: cover`).
    *   **`.card-page__title`** — Заголовок, прижатый к низу карточки. Цвет текста белый для контраста.
    *   **`.card-page__link`** — Ссылка-перекрытие. Она пустая внутри, но растянута на всю площадь родителя (`inset: 0`), чтобы клик в любое место карточки вел на статью.
